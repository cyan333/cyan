Tape-out 1 - 2019/04/22

Advanced Encryption Standard (AES) Test Chip

The Advanced Encryption Standard, or AES, is a symmetric block cipher chosen by the U.S. government to protect classified information and is implemented in software and hardware throughout the world to encrypt sensitive data.

This test chip focuses on creating a more sercure solution for the AES block and is my pre-Ph.D. research project. This project is completed by 2 other teammates and me. My task in this test chip is mainly focus on the analog blocks and a small dummy control logic.

Tools

  1. Cadence Virtuoso - for schematic and layout
  2. Design Vision - digital block simulation
  3. Cadence ADE L - for analog block simulation

Programming languages

  1. Verilog - for digital block
  2. C language - for testing

Details

  • Work with the team to create communication plan between analog and digital blocks
  • Establish and implement in Cadence Virtuoso
  • Evaluating silicon performance and simulate the circuit in Cadence ADE L
  • Layout the analog block in Cadence
  • Validate circuit performance in the lab using microprocessor
